FAANG Stock Price Analysis & Prediction

AI & Data Analytics Internship-Oriented Project

Overview

This project demonstrates a complete Artificial Intelligence and Data Analytics workflow applied to financial time-series data. It focuses on analyzing historical stock prices of FAANG companies and predicting next-day closing prices using technical indicators and machine learning models.

Problem Statement

Financial markets generate high-volume and time-dependent data. Extracting insights and building predictive systems require systematic data analysis and machine learning techniques.

Analyze historical FAANG stock price trends
Engineer technical indicators used in quantitative finance
Predict short-term stock price movements

Dataset

Type: Historical daily stock market data
Format: CSV
Companies: AAPL, AMZN, GOOGL, META, MSFT

Core Features

Date, Ticker, Open, High, Low, Close, Volume

Engineered Features

Moving Averages: SMA_7, SMA_21, EMA_12, EMA_26
Momentum Indicators: RSI_14, MACD, MACD_Signal
Volatility Indicators: Bollinger Bands, Volatility_7d
Target Variable: Next_Day_Close

Methodology

Data Preprocessing: cleaning, formatting, handling missing values
Exploratory Data Analysis: trend, volume, and volatility analysis
Feature Engineering: computation of technical indicators
Model Training: machine learning-based prediction
Evaluation: accuracy and error metrics
